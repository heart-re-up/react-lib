# @heart-re-up/react-lib

React 라이브러리입니다. 재사용 가능한 커스텀 훅 및 컨텍스트/프로바이더들을 제공합니다.

## 설치

````bash
npm install @heart-re-up/react-lib
# 또는
yarn add @heart-re-up/react-lib
# 또는
pnpm add @heart-re-up/react-lib
```Ø

## 개발

```bash
# 의존성 설치
pnpm install

# 개발 모드로 빌드 (watch)
pnpm build:watch

# 테스트 실행
pnpm test

# 테스트 UI
pnpm test:ui

# 커버리지 확인
pnpm test:coverage

# 린트 검사
pnpm lint

# 포맷팅
pnpm format
````
